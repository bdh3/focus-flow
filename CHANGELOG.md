# 📝 Focus Flow 변경 이력 (Changelog)

모든 버전의 상세 변경 내역을 기록합니다. `README.md`는 최신 버전의 요약 정보만 제공합니다.

---

## [v1.8.2-hotfix] - 2024-05-23
### Fixed
- **전체 화면 알람 돌파 실패 해결**: 화면이 켜져 있을 때 배너(HUN) 중복을 피하기 위해 중요도를 낮췄던 로직이 `fullScreenIntent` 실행을 차단하던 문제 수정. 무조건 `IMPORTANCE_HIGH`를 유지하되 `setOnlyAlertOnce`로 배너 중복 억제.

---

## [v1.8.2] - 2024-05-23
### Added
- **설정 보호 UX**: 타이머 진행 중(`isTimerActive`) 집중 방식, 시간 설정, 소리/진동 패턴 등 실행 중인 타이머에 영향을 주지 않는 UI 항목들을 비활성화(Disabled).
### Fixed
- **알람 정시성 강화**: Doze 모드 대응을 위해 `AlarmManager.setAlarmClock` 적용 및 `WakeLock` 중첩 획득 로직 개선.
- **중복 알림 배너 제거**: 전체 화면 알람 시 시스템 상단 배너가 액티비티를 가리지 않도록 알림 카테고리 및 채널 분리.
- **사운드 지연 튜닝**: 액티비티 전환 시 벨소리 끊김 방지를 위해 화면 상태별 지연 시간(400ms/200ms) 정밀 조정.

---

## [v1.8.1] - 2024-05-22
### Fixed
- **커스텀 작업 설정 무시 버그**: 스케줄 생성 시 개별 집중/휴식 시간이 전역 설정에 의해 덮어씌워지는 현상 수정.
- **타이머 세션 고정**: 세션 시작 시점의 설정값이 실행 중에 변하지 않도록 상태 관리 안정성 확보.

---

## [v1.8.0] - 2024-05-20
### Added
- **브랜드 용어 개편**: '몰입' -> '집중', '독립 세션' -> '집중 세션'으로 명칭 변경.
- **알람 UI/UX**: 중단 버튼 확대(80dp), Midnight Blue 배경 고정.
- **설정 디자인**: 뽀모도로 프리셋(25/5, 50/10) 카드 스타일 리뉴얼.
- **Z 플립 5 호환**: 접힌 상태/잠금 화면 위로 알람 노출 로직 강화.
