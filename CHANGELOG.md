# 📝 Focus Flow 변경 이력 (Changelog)

모든 버전의 상세 변경 내역을 기록합니다. `README.md`는 최신 버전의 요약 정보만 제공합니다.

---

## [v1.8.3] - 2026-04-18
### Fixed
- **캘린더 네비게이션 동기화 (Patch 16)**: `HorizontalPager` 애니메이션 중 중간 날짜가 ViewModel을 오염시키는 현상을 `isScrollInProgress` 가드 및 `scrollToPage` 즉시 이동으로 해결.
- **날짜 계산 로직 강화 (Patch 7)**: 타임존 오프셋에 의한 날짜 밀림을 방지하기 위해 선택된 블록의 `minOrNull()`을 최우선 시작 시점으로 채택.
- **자정 경계선 표시 개선 (Patch 8)**: `ScheduleDao` 중첩 쿼리 적용으로 자정 전후에 걸친 일정이 양일 캘린더에 모두 노출되도록 수정.
- **데이터 동기화 및 시간 표시 오류 해결 (Patch 9, 13)**: 일정 생성/로드 시 `durationMinutes` 전달 누락을 수정하여 "19:00 ~ 19:00" 등 시간 표시 오류 해결.
- **세션 고정(Session Locking) 적용 (Patch 14)**: 타이머 활성화 시 설정 탭의 기본값이 현재 작업 정보를 덮어쓰지 못하도록 차단.
- **UI 반응성 및 디자인 정리 (Patch 10, 11, 15)**: 설정 변경 시 타이머 즉시 갱신, 연속 모드 전체 시간 설정 허용, 불필요한 가이드 문구 제거.
- **캘린더 퀵 세션 생성 버그 수정 (Patch 17)**: 설정된 휴식 시간(사이클)이 무시되고 연속 집중 모드로 고정되어 생성되던 현상 해결.
