# Focus Flow 알람 시스템 명세 (v1.7.3)

본 문서는 갤럭시 알람 스타일의 고도화된 사운드/진동 시스템 및 알람 동작 정책을 정의합니다.

## 1. 알람 트리거 및 제외 정책
- **최초 시작 시 제외**: 타이머 세션을 유저가 직접 처음 시작할 때(Start 버튼 클릭 시)는 알람이나 팝업을 띄우지 않습니다. 
  - *예외: 추후 '자동 스케줄 시작' 기능 도입 시, 해당 시점에는 몰입 시작 알람을 발생시킵니다.*
- **넘기기(Skip) 동작**: 유저의 수동 행동이지만, 유저가 **'전체 화면 알람'**을 설정했거나 사운드가 **벨소리(Ringtone)**인 경우 설정을 존중하여 **전체 화면 알람**을 표시합니다. 그 외에는 상단 팝업으로 표시합니다.

## 2. 알람 모드 및 사운드 구성
- **알림 방법 (3종)**: 
  - 벨소리(Ringtone), 알림음(효과음), 진동(Vibration)
- **알람 노출 모드 (2종 유저 선택)**:
  1. **전체 화면 알람**: 기기 잠금/미사용 시에도 화면 전체 점유. (기본값)
  2. **상단 팝업 알람**: 화면 상단 헤드업 알림.

### [알람 노출 모드 결정 트리 (v1.7.3 핵심 규칙)]
1. **벨소리 우선 법칙**: 집중/휴식/종료 개별 사운드가 **벨소리(Ringtone)**로 설정된 경우, 설정 탭의 노출 모드(팝업/전체화면)와 무관하게 **무조건 전체 화면 알람**으로 작동합니다.
2. **전체 화면 설정 존중**: 설정 탭에서 **'전체 화면 알람'**을 선택한 경우, 사운드가 알림음이나 진동일지라도 **전체 화면 알람**으로 작동합니다. (넘기기 시에도 적용)
3. **팝업 설정 및 종료 알람**: 설정 탭에서 **'상단 팝업 알람'**을 선택한 경우, 사운드가 벨소리가 아니라면(알림음/진동) **종료 알람을 포함한 모든 상황에서 상단 팝업**으로 작동합니다.
4. **매너 모드 준수**: 시스템이 무음/진동 모드인 경우, 소리 설정이 켜져 있더라도 소리를 내지 않고 진동과 화면 노출 규칙만 따릅니다.

| 사운드/진동 설정 | 설정 탭 노출 모드 | 최종 노출 결과 | 비고 |
| :--- | :--- | :--- | :--- |
| **벨소리** | 전체 화면 OR 팝업 | **전체 화면** | 규칙 1 적용 |
| **알림음/진동만** | 전체 화면 | **전체 화면** | 규칙 2 적용 |
| **알림음/진동만** | 팝업 알람 | **상단 팝업** | 규칙 3 적용 (종료 알람 포함) |

## 3. 세션별 알람 패턴 및 문구
알람은 설정 탭에서 지정된 **집중, 휴식, 종료** 사운드가 각각 반영되어야 합니다.

> **규칙**: `{작업명}`을 지정하지 않은 1회성 세션의 경우, `{작업명}` 자리에 **"독립 세션"**이라는 명칭을 사용합니다. (예: `독립 세션: 몰입 시작`, `독립 세션: 휴식 시작`)

| 패턴 | 상황 | 알람 타이틀 (팝업/전체 공통) | 알람 내용 (전체 화면 전용) |
| :--- | :--- | :--- | :--- |
| **몰입 시작** | 휴식 종료 후 몰입 진입 | `{작업명}: 몰입 시작 (경과 시간)` | "흐름을 타고 집중을 시작할 시간입니다." |
| **휴식 시작** | 몰입 종료 후 휴식 진입 | `{작업명}: 휴식 시작 (경과 시간)` | "잠시 숨을 고르며 에너지를 충전하세요." |
| **몰입(연속)** | 이전 몰입 종료 후 다음 몰입 진입 | `{작업명}: 몰입 (경과 시간)` | "계속해서 집중을 유지해 보세요." |
| **종료** | 전체 세션 완료 | `{작업명}: 종료 (경과 시간)` | "모든 세션을 완료했습니다. 수고하셨습니다!" |

- **경과 시간 표시 형식**:
  - 60분 미만: `(25분 경과)`
  - 60분 이상: `(1시간 10분 경과)`
  - *단, 0분(최초 시작)인 경우 경과 시간을 표시하지 않습니다.*

## 4. 사용자 경험(UX) 및 시스템 로직
- **단일 알림 정책 (v1.7.3)**: 
  - 중복 팝업 방지를 위해 기존 서비스 알림(ID 1000)을 업데이트하는 방식으로 통합되었습니다. (별도의 1001번 알림 생성 안 함)
  - 전체 화면 모드 시 시스템의 자동 팝업을 억제하기 위해 `PRIORITY_LOW` 및 `CATEGORY_SERVICE` 조합을 사용합니다.
- **사운드 정지 및 서비스 종료**: 
  - 사용자가 알람창에서 **[중단]** 버튼을 클릭하거나 앱에 진입하면 즉시 소리/진동이 멈춥니다.
  - 종료 알람 시, 벨소리 생명주기 보장을 위해 서비스는 25초간 대기 상태를 유지하지만, 사용자가 **[중단]**을 누르는 즉시 대기를 취소하고 세션을 완전히 종료합니다.
- **정보 표시**:
  - **전체 화면**: 타이틀 + 감성 문구 + 작업 경과 시간.
  - **상단 팝업**: 타이틀 + 작업 경과 시간 (내용 생략으로 간결함 유지).
- **20초 타임아웃**: 유저 응답이 없어도 **최대 20초** 후 사운드와 진동이 자동 종료되며, 5초 후 서비스도 자동 종료됩니다.
- **타이머 지속성**: 알람 화면이 떠 있는 동안에도 타이머는 멈추지 않고 흐름을 유지합니다.
