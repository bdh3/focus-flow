# 🌊 포커스 플로우 (Focus Flow)

> **"시간을 블록으로 그리다, 몰입을 흐름으로 만들다."**

**Focus Flow**는 막연하게 흘러가는 시간을 눈에 보이는 **'15분 단위 블록'**으로 설계하여, 당신의 집중력을 완벽한 흐름(Flow)으로 만들어주는 ADHD 특화 시각적 스케줄러입니다.

---

## 📖 초보자 가이드: 시간을 지배하는 블록 활용법

### 1. 📅 계획하기 (시간을 블록으로 그리다)
- **블록 선택 (앵커 시스템):** 시작 시간을 터치한 후 종료 시간을 터치하세요. 사이의 블록이 자동으로 연결됩니다.
- **범위 수정:** 중간 블록을 터치해 길이를 바로 조절하거나, 첫 블록을 다시 터치해 전체를 취소할 수 있습니다.
- **세션 생성:** `세션 생성` 시 타이머 탭으로 자동 이동하여 즉시 몰입 준비가 완료됩니다.

### 2. ⏱️ 실행하기 (감각적인 몰입의 시작)
- **지능형 구간 알림:** 설정한 주기(기본 15분)마다 팝업 알림이 뜨며 "현재 무엇을 하고 있는지"를 계속 상기시켜줍니다.
- **넘기기(Skip):** 작업이 일찍 끝났다면 `넘기기`를 눌러 다음 구간으로 즉시 점프하세요.
- **상태바 고정:** 앱을 닫아도 알림창에서 현재 작업명과 남은 시간을 한눈에 확인할 수 있습니다.

### 3. 📋 관리하기 (일목요연한 스케줄)
- **시간순 자동 정렬:** 모든 작업은 수행 시간 순서대로 정렬되어 다음에 할 일을 고민할 필요가 없습니다.
- **완벽한 동기화:** 캘린더와 작업 리스트가 실시간으로 연동되어 데이터 중복 없이 깔끔하게 관리됩니다.

## ✨ 주요 기능 (v1.2.5)

- **[NEW] 작업 선택 토글 해제**: 타이머 탭의 작업 리스트에서 이미 선택된 작업을 다시 누르면 선택이 해제되어 초기 "작업을 선택하세요" 상태로 돌아갑니다.
- **[NEW] 내비게이션 레이어 겹침 버그 수정**: 탭 전환 시 백스택을 초기화하여 화면이 겹치거나 이전 화면이 남는 현상을 해결했습니다.
- **[NEW] 앵커 기반 블록 선택**: 처음과 끝을 지정하고 중간을 터치해 범위를 자유롭게 수정하는 직관적인 UX.
- **[NEW] 세션 생성 후 자동 전환**: 캘린더에서 계획을 세우면 즉시 타이머 모드로 로드되어 시작 대기 상태가 됩니다.
- **[NEW] 작업 리스트 정렬 및 싱크**: 시간대별 자동 정렬과 스케줄 수정 시 실시간 반영 로직이 강화되었습니다.
- **지능형 알림 시스템**: 
    - 세션 종료 시 "몰입 완료! 🎉" 팝업 및 강력한 진동 보상.
    - 포그라운드 서비스를 통한 백그라운드 타이머 안정성 확보.
- **사용자 맞춤 설정**: 알람 주기(5/15/30분) 및 진동 온/오프 기능을 제공합니다.

## ⚙️ 기본 설정 (Default Settings)

- **알림 단위**: 15분 (사용자 변경 가능)
- **진동**: 활성화 (공공장소용 Off 기능 제공)
- **버전**: v1.2.4 (Code 20)
