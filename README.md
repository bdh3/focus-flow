# 포커스 플로우 (Focus Flow)

**Focus Flow**는 집중의 흐름을 유지하고 시간을 시각적으로 관리하는 안드로이드 앱입니다. 15분 단위의 블록 시스템과 포그라운드 서비스를 통해 사용자가 설정한 리듬에 맞춰 끊김 없는 몰입(Flow)을 유지하도록 돕습니다.

## 🌟 핵심 요약 (4-Line Description)

1.  **작업 중심의 블록 몰입**: 선택한 할 일을 15분 단위 블록으로 구성하여 깊이 몰입할 수 있도록 돕는 사이클 타이머입니다.
2.  **포그라운드 서비스 기반 안정성**: 앱이 백그라운드에 있거나 화면이 꺼져도 타이머가 중단되지 않고 정확한 알림을 제공합니다.
3.  **단일 타이머 원칙**: 몰입의 분산을 막기 위해 한 번에 오직 하나의 작업(Task)만 수행 가능하며, 실행 중에는 다른 작업으로의 전환이 제한됩니다.
4.  **시각적 스케줄링**: 캘린더 뷰에서 15분 단위로 일정을 직접 그리고, 완료된 세션은 시스템 캘린더에 자동으로 기록됩니다.

## ✨ 주요 기능 (v1.1.8)

- **포그라운드 타이머**: 시스템 서비스(Foreground Service)를 도입하여 백그라운드에서도 타이머와 알림이 안정적으로 유지됩니다.
- **캘린더 일간/월간 뷰**: 상단 바에서 뷰를 전환할 수 있으며, 일간 뷰 진입 시 현재 시간으로 자동 스크롤됩니다.
- **15분 블록 레이아웃**: 1시간을 4개의 가로 블록으로 배치하여 화면 시인성을 높이고 효율적으로 시간을 관리합니다.
- **범위 제어 시스템**: 클릭 한 번으로 여러 블록을 채우거나(Fill), 이미 선택된 범위 내를 클릭하여 이후 범위를 축소(Shrink)할 수 있습니다.
- **작업(Task) 보호**: 타이머가 실행 중인 경우, 현재 진행 중인 작업 외의 다른 작업 수정이나 새로운 세션 시작이 방지되어 '한 번에 하나'의 몰입을 강제합니다.
- **캘린더 자동 연동**: 집중 세션 완료 시 삼성/구글 캘린더에 정확한 시작/종료 시간과 함께 기록을 남깁니다.
- **단일 집중 모드**: '휴식' 개념을 제거하고 오직 '집중'에만 최적화된 단순하고 강력한 흐름을 제공합니다.
- **커스텀 햅틱 피드백**: 집중 종료 시 묵직한 진동 패턴을 통해 명확한 완료 인지를 돕습니다.

## ⚙️ 기본 설정 (Default Settings)

- **알림 단위**: 15분 (5/15/30분 중 선택 가능)
- **진동**: 활성화 (앱 설정에서 제어 가능)
- **기본 타이머**: 1시간 (60분)
- **기본 작업명**: 입력하지 않을 시 'Task1'로 자동 지정
