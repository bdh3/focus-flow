# ADHD 블록 스케줄러 (ADHD Block Scheduler)

집중력 유지가 어려운 분들을 위한 블록 기반의 시간 관리 안드로이드 앱입니다. 뽀모도로 기법과 유사하지만, 사용자가 자신의 리듬에 맞게 시간을 설정하고 시각적으로 관리할 수 있도록 돕는 데 중점을 둡니다.

## 🎯 주요 목표

ADHD 성향을 가진 사용자들이 시간의 흐름을 직관적으로 인지하고, '집중'과 '휴식'의 경계를 명확히 하여 번아웃 없이 꾸준히 작업을 이어갈 수 있도록 돕습니다.

## ✨ 주요 기능

- **블록 기반 시간 관리**: 1시간을 기본 단위로 하여 4개의 블록(1블록 = 15분)으로 나눕니다.
- **45/15 사이클**: 기본적으로 3개의 블록(45분)은 '집중 시간'으로, 마지막 1개의 블록(15분)은 '휴식 시간'으로 설정됩니다.
- **커스터마이징 가능한 시간 설정**: 사용자는 자신의 필요에 따라 블록의 길이와 개수, 집중/휴식 시간의 비율을 자유롭게 조절할 수 있습니다.
- **시각적 인터페이스**: 시간의 흐름을 색상이나 채워지는 형태로 시각화하여 남은 시간을 직관적으로 파악할 수 있습니다.
- **작업(Task) 관리**: 각 집중 블록에 수행할 작업을 할당하고 완료 여부를 체크할 수 있습니다.
- **알림 기능**: 각 블록의 시작과 끝, 휴식 시간의 시작을 알려주어 다른 활동에 몰입했더라도 스케줄로 돌아올 수 있도록 돕습니다.
- **캘린더 연동**: 생성된 집중 블록을 사용자의 기기 캘린더(삼성 캘린더 포함)에 자동으로 기록하여 시간 사용 내역을 관리합니다.
- **간단한 통계**: 하루 동안의 집중 시간, 완료한 작업 개수 등을 보여주는 간단한 리포트를 제공합니다.

## 🛠️ 기술 스택 및 개발 환경

- **Platform**: Android
- **Language**: Kotlin
- **UI**: Jetpack Compose
- **Architecture**: MVVM (Model-View-ViewModel)
- **Asynchronous**: Coroutines
- **Local DB**: Room (작업, 설정 등 저장)

## 🚀 빌드 및 실행 방법

1.  **Prerequisites**: Android Studio (최신 버전 권장), JDK 11 이상
2.  `git clone https://github.com/bdh3/adhd-block-scheduler.git`
3.  Android Studio에서 프로젝트를 엽니다.
4.  필요한 Gradle 종속성이 동기화될 때까지 기다립니다.
5.  에뮬레이터 또는 실제 기기에서 앱을 실행합니다.
