# 🌊 포커스 플로우 (Focus Flow)

> **"시간을 블록으로 그리다, 몰입을 흐름으로 만들다."**

**Focus Flow**는 막연하게 흘러가는 시간을 눈에 보이는 **'15분 단위 블록'**으로 설계하여, 당신의 집중력을 완벽한 흐름(Flow)으로 만들어주는 ADHD 특화 시각적 스케줄러입니다.

---

## 📖 초보자 가이드: 시간을 지배하는 블록 활용법

### 1. 📅 계획하기 (시간을 블록으로 그리다)
- **블록 선택 (앵커 시스템):** 시작 시간을 터치한 후 종료 시간을 터치하세요. 사이의 블록이 자동으로 연결됩니다.
- **범위 수정:** 중간 블록을 터치해 길이를 바로 조절하거나, 첫 블록을 다시 터치해 전체를 취소할 수 있습니다.
- **세션 생성:** 세션을 생성하면 타이머 탭으로 자동 이동하며, 데이터가 로드된 '대기' 상태가 됩니다.

### 2. ⏱️ 실행하기 (감각적인 몰입의 시작)
- **사용자 수동 시작 (v1.3.1):** 스케줄 로드 후 사용자가 준비되었을 때 직접 '시작' 버튼을 눌러 몰입을 시작합니다.
- **시스템 레벨 정밀 알람:** `AlarmManager`를 도입하여 백그라운드나 Doze 모드에서도 정확한 시점에 구간 알림을 제공합니다.
- **넘기기(Skip):** 작업이 일찍 끝났다면 `넘기기`를 눌러 다음 구간으로 즉시 점프하세요.

### 3. 📋 관리하기 (일목요연한 스케줄)
- **배터리 최적화 대응:** 설정에서 배터리 최적화 예외를 요청하여, 장시간 타이머 구동 시 시스템에 의해 앱이 종료되는 것을 방지할 수 있습니다.
- **완벽한 동기화:** 캘린더와 작업 리스트가 실시간으로 연동되어 데이터 중복 없이 깔끔하게 관리됩니다.

## ✨ 주요 기능 (v1.3.1)

- **[NEW] 진동 설정 고도화**: 듀얼 알림 채널 도입으로 시스템 설정에 구애받지 않고 앱 내 설정이 실시간 반영되도록 개선.
- **[NEW] 카운트다운 소실 방지**: 넘기기(Skip) 시 1초가 사라지던 버그를 해결하고 정수 초 보정 로직 적용.
- **[NEW] 작업 선택 잠금**: 타이머 실행 중 타 작업으로의 이탈 및 스케줄 꼬임 방지 로직 강화.
- **고정밀 타이머 시스템**: `AlarmManager.setExactAndAllowWhileIdle` 적용으로 시스템 레벨의 알람 정확도 확보.
- **지능형 알림 시스템**: 
    - 세션 종료 및 구간 전환 시 진동/팝업 보상.
    - 포그라운드 서비스를 통한 실시간 상태바 타이머 제공.
- **앵커 기반 블록 선택**: 처음과 끝을 지정하는 직관적인 범위 지정 방식.

## ⚙️ 기본 설정 (Default Settings)

- **알림 단위**: 15분 (사용자 변경 가능)
- **진동**: 활성화 (공공장소용 Off 기능 제공)
- **최신 버전**: v1.3.1 (Code 27)
